# 최종 코딩 테스트

---

## 📜 구현 기능 목록

### 라인

- 라인은 이름과 역 레파지토리를 가진다.

### 역
- 역은 이름을 가진다.

### 비즈니스 로직

- 초기 설정
  - 역, 노선, 구간 정보를 등록
- 최단 거리를 찾기
- 최소 시간을 찾기

### 입/출력

### 입력

- 메인 화면 입력
- 경로 기준 입력
- 출발역, 도착역 입력

### 출력

- 기능 선택 출력
- 조회 결과 출력
- 기대하는 출력 시 `[INFO]`를 붙여 출력
- 예외 발생 시 오류 메시지 출력
  - `[ERROR]`로 시작해야한다.

### ⚠️ 예외 처리

잘못된 값을 입력할 경우 `IllegalArgumentException`를 발생시키고, `[ERROR]`로 시작하는 에러 메시지를 출력 후 경로 기준 입력을 다시 받는다.

- 유효성 검사 실패 시 예외 발생
